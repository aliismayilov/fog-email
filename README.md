# Fog::Email

This gem aims to abstract usage of popular email service providers
such as Sendgrid, Mandril, Amazon SES and etc.

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'fog-email'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install fog-email
